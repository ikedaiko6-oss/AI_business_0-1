# B4 顧客リアリティチェック ── 「遠方相続人」は本当にいるのか？を実データで検証

作成日: 2026-07-02
きっかけ: 「そんな境遇の人、あんまりいなくない？ 親が死ぬくらいだから60代くらいだよね？」という指摘。
これまでのリサーチは**制度側（法改正・市場規模・競合）ばかり厚く、顧客側（誰が本当に困っているか）を検証していなかった**。ここで一次データに当たり直した。

---

## 0. 結論（先に言う）

**指摘は正しかった。** 元のペルソナ「東京在住50代サラリーマン、遠い実家を見に行けない」は**少数派**。データが示す本当の空き家所有者は：

> **60〜75歳以上。空き家まで1時間以内に住んでいて、自分や親族でときどき見に行っている。**
> 困りごとは「見に行けない」ではなく、**「体がきつい」「どうするか決められない」「金をかけたくない」**。

さらに、「税金6倍の恐怖で駆け込む人」も現時点ではほぼ存在しない（**勧告は全国でまだ378件**）。

→ **B4を「あなたの代わりに見てきます（診断レポート）」のまま売ると外す可能性が高い。** ただし死ぬわけではない。刺さる形に3つ再定義できる（§3）。即金性の評価は下方修正が誠実（§5）。

## 3. では誰が客なのか ── 生き残る3セグメント

データから逆算すると、本当にいる「困った人」は3種類。

### P1. 「行けるけど、もう体が動かない」層（最大ボリューム）

- **誰**：75歳以上が39%。空き家は近い。見には行ける。**でも草刈り・庭木・雨樋掃除・片付けの体力がない**。
- **本当の困りごと**：「見る」ことではなく「**作業**」。
- **買うもの**：**草刈り・庭木・簡易修繕などの実作業**＋「ちゃんと管理してます」の記録（写真つき管理記録＝行政や近隣への証跡）。
- **含意**：**B4の主従逆転**。診断レポートは主商品ではなく、**作業に付いてくる差別化のおまけ**にする。単価は作業で取る。
- **競合**：便利屋・シルバー人材と正面衝突。差別化は「管理記録・行政対応の証跡が残る」＋固定客化（年間の手入れ暦）。

### P2. 「どうするか決められない」層（33.1%＝質が高い）

- **誰**：相続したが、解体費・思い出・共有名義・手続きで**フリーズしている**60代の子世代。
- **本当の困りごと**：情報がバラバラで**意思決定ができない**（売る？壊す？貸す？国庫帰属？いくらかかる？）。
- **買うもの**：**「決めるための材料一式」**＝現地の状態＋解体概算＋売却査定の取次＋選択肢の比較を1枚に整理したレポート。
- **含意**：これは元のB4（初動レポート）が**唯一そのまま生きる**セグメント。ただし売り方は「不安訴求」ではなく「**前に進める材料**」。買い手は本人よりも、案件を前に進めたい**司法書士・不動産屋**（彼らの商談が止まる原因が「現地と費用感が分からない」）。
- **単価**：¥30,000〜50,000が通る唯一の層。

### P3. 「指導が来ちゃった」層（小さいが激アツ・今後増える）

- **誰**：自治体から指導を受けた3,211件（＋これから増える）。**放置の言い訳が切れた人**。
- **本当の困りごと**：「何をどこまで直せば勧告を回避できるか」＋是正の実働。
- **買うもの**：是正計画＋実作業＋**是正完了の記録レポート**（自治体に見せる用）。
- **チャネル**：**自治体の空き家窓口一択**。「指導を出した後、所有者に紹介できる是正業者リストはありますか？」と聞きに行く。
- **含意**：件数は小さいが確度・単価が高く、**勧告378→数千件に増えていく成長株**。先にポジションを取る価値がある。

## 4. B4の再定義

| | 旧B4（検証前） | **新B4（データ反映後）** |
|---|---|---|
| 主商品 | 現況診断レポート | **現場作業（草刈り・手入れ・片付け手配）＋管理記録** |
| レポートの位置づけ | 主役 | **差別化のおまけ＆P2/P3向けの武器** |
| 主な客 | 遠方の50代相続人 | **近くに住む60〜80代所有者（P1）／決められない子世代（P2）／指導を受けた人（P3）** |
| 売り文句 | 「代わりに見てきます」 | 「**手入れも記録もまとめて面倒見ます**」「**どうするか決める材料を1枚にします**」 |
| 恐怖訴求（税6倍） | 前面に出す | **使わない**（まだ現実になっていない。指導を受けたP3にだけ有効） |
| 市場の時間軸 | 今すぐ大きい | **今は小さめ、2040年に向けて構造的に拡大** |

**強み適合はむしろ上がる**：作業が主になるほど「山開拓の体力・機材」が活き、記録・レポートのAI量産は差別化として残る。二刀流の構図は崩れない。**崩れたのは「レポート単品で高く売れる」という想定**。

## 6. 電話で確かめることも変わる（ヒアリング質問の更新）

| 相手 | 旧質問 | **新質問** |
|---|---|---|
| 司法書士・税理士 | 「遠方で現地を見られない案件ある？」 | 「**実家をどうするか決められずに止まっている相続案件、現地の状態と費用感の資料があれば前に進むもの、ありますか？**」 |
| 自治体 空き家窓口 | 「現況調査の外注ニーズある？」 | 「**指導を出した後、所有者から『どうすればいい』と相談が来ませんか？ 紹介できる是正業者のリストはありますか？**」 |
| シルバー人材・便利屋 | （競合として見ていた） | 「**空き家の草刈り依頼って全体の何割です？ 断ってる案件あります？**」（P1の実在量の確認＋下請け/提携の探り） |
| 不動産屋 | 「査定前の現況把握ニーズ？」 | 「**売主が決めきれずに塩漬けの物件、背中を押す材料（現地＋費用比較）があれば動きますか？**」 |

**電話10件の判定基準は同じ（3件で継続、5件で熱い）が、聞くべき中身がこれで正しくなった。**

## 7. 教訓（次のリサーチのために）

1. **制度・競合・市場規模をいくら調べても、「困った人が実在するか」は別の調査。** 顧客側の一次データ（実態調査・アンケート）を最初に当てる。
2. **「法律が変わった＝困る人が生まれる」ではない。** 執行された件数（勧告378件）を見るまで需要と呼ばない。
3. **ペルソナは想像で作らず、年齢分布・距離・行動データから逆算する。**
4. この3点は、C群（森林・ため池）にも同じ検証が必要（→ 次回の宿題：**市町村側の「発注実績」の実額確認**。Week1の電話で聞く）。
