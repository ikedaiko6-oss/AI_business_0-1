## 今の結論（2026-06-30）

強みは「**現地に行って記録し、遠隔の意思決定者が判断できるAIレポートにする**」1つの能力。
これが刺さる供給ギャップ市場の二本柱：

1. **野立て低圧太陽光の O&M（草刈り＋点検＋AIレポート）** … 点検が法的に義務＝逃げない需要 × O&M人材枯渇
2. **相続・遠隔地の空き家＋山林の初動・判断レポート** … 空き家900万戸・相続が主因・登記義務化

**市場規模（TAM）で一番でかい山も太陽光O&M（再エネO&M 2030年度 約1.5兆円）。** 空き家は"問題の総量"は最大だが課金できる塊は細い。

**資格はほぼ不要で開始できる**：どの案件も「現地に行って記録しレポートにする」本体はノー資格。資格/許可が要るのは①電気工事②廃棄物運搬③捕獲④士業の独占業務だけで、そこは提携に回す。まず要るのは資格より「損害賠償責任保険」と「提携先」。

詳細・出典は research/ の各レポート参照（§3 市場規模ランキング／§4 資格・許認可マップ を追記済み）。
