## 3. 無料デモ点検チェックリスト（現地の型・再現性の核）

**所要目安: 1件45分（未検証・要実測）**

**A. 到着時（5分）**
- [ ] 全景を4隅から撮影（同じ高さ・同じ向きで。自撮り棒で画角統一）
- [ ] 入口・フェンス・標識の状態を撮影（破損・施錠の有無）

**B. 外観点検（15分）**
- [ ] 雑草の高さをパネル下端と比較して撮影(影がかかっているか)
- [ ] パネル表面: 汚れ・鳥フン・破損・ホットスポットの兆候を目視
- [ ] 架台・基礎: 錆・傾き・地面の陥没
- [ ] フェンス・ケーブル: 垂れ下がり・小動物の侵入痕

**C. サーマル撮影（15分）**
- [ ] 晴天時に実施（曇天はホットスポットが出にくい）
- [ ] パネル列ごとに正面からサーマル+可視光をセットで撮影
- [ ] 温度差が出た箇所は近接で再撮影（マニュアルフォーカスで合わせる）
- [ ] パワコン・接続箱の外観をサーマルで撮影（**蓋は開けない**＝電気工事士領域に踏み込まない）

**D. 退出時（5分）**
- [ ] 施錠・原状回復を確認
- [ ] その場でオーナー/担当者に「速報1枚」（気になった箇所の写真だけ）をLINE/メールで送る → 本レポートは当日夜にAI生成して送付

**禁止ライン(常に)**: 蓋開け・配線接触・活線近接はしない。異常の断定はせず「専門家(電気主任技術者)に確認すべき点」として書く。

## 4. AIレポート生成プロンプト（たたき台v1）

当日夜、撮影した写真をClaude/ChatGPTに読ませて使う。出力は `sample-report-solar.html` の構成に合わせる。

```
あなたは太陽光発電所の現地点検レポート作成アシスタントです。
添付の写真(全景・外観・サーマル画像)と以下のメモから、遠隔地のオーナーが
1枚で状況判断できる点検レポートを日本語で作成してください。

# 現場メモ
- 発電所名/所在地: ◯◯
- 点検日時/天候: ◯◯
- 気づいた点(自由記入): ◯◯

# 出力ルール(必ず守る)
1. 構成: ①総合サマリ(3行) ②写真ごとの所見 ③雑草・外観・サーマルの3区分評価(良好/経過観察/要確認の3段階) ④次回までの確認事項
2. 断定しない: 故障・不具合と断定せず「〜の可能性があるため、電気主任技術者等の専門家に確認を推奨」と書く
3. 「推奨・修理すべき」という表現は使わず「確認事項/選択肢/専門家に相談すべき点」として書く
4. 専門用語には一言の説明を付ける(オーナーは非専門家)
5. A4で1枚に収まる分量
```

**運用メモ**: デモ2〜3件でプロンプトを回したら、出力のブレ(評価基準・言い回し)を見て v2 に改訂し、このファイルに追記していく。
