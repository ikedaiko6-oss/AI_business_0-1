# S1 太陽光O&M の差別化設計（ガジェット武装型・機材リサーチ）

作成日: 2026-07-04 ／ 前提: [`S1-solar-om-playbook.md`](S1-solar-om-playbook.md)の具体化。低圧太陽光（50kW未満）の点検を対象

## 0. ここに至った経緯（要約）

「デジタル×アナログ何でも屋」を検討したが、見守り市場全体が激戦区と判明し見送り（[`2026-07-04-digital-analog-benriya-research.md`](2026-07-04-digital-analog-benriya-research.md)）。北原孝彦氏の型（①1本に絞る②再現性③供給側のボトルネックを突く）を当てはめ直した結果、既存の本命候補であるS1（太陽光O&M）に戻ることにした。

ただし、単なる「草刈り屋」だと本人にとって作業が単調（楽しさの観点で弱い）という課題があったため、**ドローン・サーマルカメラ・AIレポートというガジェット要素を中心に据えた設計**に組み替える。あわせて、クリス・コーナーの型（[`2026-07-04-business-model-database.md`](2026-07-04-business-model-database.md)のモデルA/B/C）を具体的な運用フェーズに組み込む。

## 1. コンセプト（2フェーズ）

### Phase 1: 自分で低圧太陽光O&Mを「AI武装」でやる（モデルA: コピー戦略の"自分で検証"部分）

- 現地訪問→タブレット/ハンディサーマルカメラで撮影→AIでレポート自動生成、という型を自分でまず回し、実績（成功事例）を作る
- 草刈り自体は外注前提（提携業者・シルバー人材センター等）にし、自分の作業は「点検データの取得」と「AIレポート化」に絞る
- クリス・コーナーの**モデルB（小さくテスト）**をそのまま適用: 最初から高額な機材を買わず、安価なハンディ型サーマルカメラで数件試し、契約が取れて収益化できると確認できてから機材に投資する

### Phase 2: 型ができたら、同業のO&M業者に「点検AI化ワークフロー」を売る（モデルC: AI導入代行の応用）

- Phase 1で実績ができたら、その「点検データ→AIレポート自動生成」の型を、**低圧案件で人手不足に悩む他のO&M業者**にツール／サービスとして提供する
- 営業手法はモデルCの「無料デモで即決させる」を踏襲: 相手の既存の点検現場で、自分のAIレポートと相手の従来の手書き・Excel報告書を並べて見せ、その場で品質・スピードの差を体感させる
- 課金モデルもモデルCと同じ形にできる: 導入（一括）費＋月額保守（継続課金）
- ランダムな中小企業に飛び込み営業するより、**自分が現場で信頼を積んだO&M業界内**に絞ることで、営業のハードルを下げられる（Model Cの元の弱点だった「供給側ボトルネックの根拠が薄い」を、業界を絞ることである程度補強する）

## 2. 供給ギャップの再確認

- 大規模太陽光ではAI＋ドローン点検はすでに製品化されている（センシンロボティクス「SOLAR CHECK」、エナジー・ソリューションズ「ドローンアイ」、オリックスのAIサーモ解析等）。撮影からレポート出力まで1時間で自動化する製品まである¹²
- ただし、これらは大規模発電所向けの製品。太陽光O&Mの課題として「拠点からの移動コストが経費の約18%を占める」「小規模事業所ほどデジタル化が遅れている」ことが確認されている³
- **大手のAI点検ベンダーは経済合理性から大規模発電所を狙い、低圧・小規模・散在する発電所は取りこぼされている**。この供給ギャップは[`2026-06-30-supply-gap-market-research.md`](2026-06-30-supply-gap-market-research.md)の結論と一致する

## 3. 低圧発電所の物理規模

- 10kW: パネル約34枚、130〜180㎡
- 30kW: パネル約100枚、300〜400㎡
- 50kW: パネル約143枚、基本400㎡＋法定離隔距離込みで実質600〜800㎡程度⁴
- **1件あたりテニスコート1〜2面分程度の広さ**。事業用太陽光のFIT/FIP導入量のうち10〜50kW(低圧)が容量ベースで31〜32%を占める⁵。ただし具体的な件数（基数）は、この環境のネットワークポリシー上、一次データ（fit-portal.go.jp／e-stat.go.jp）への直接アクセスができず未確認（要ローカル環境での確認）

## 4. 機材価格調査（要注意点あり）

| 選択肢 | 価格 | 備考 |
|---|---|---|
| DJI Mavic 3 Thermal（新品、DJI Care込み） | 約815,100円 | 現在は生産終了⁶ |
| DJI Matrice 4T（新品、最新機種） | 約1,108,800円 | 後継フラッグシップ⁷ |
| DJI Mavic 3 Thermal（中古・オークション相場） | 約615,500円 | 中古でも高額⁸ |
| FLIR Vue Pro R（既存ドローンへの後付けサーマルカメラ単体） | 626,000円〜 | ドローン本体は別途⁹ |
| FLIR ONE Edge Pro（スマホ装着型ハンディサーマルカメラ） | 具体的な現在価格は未確認（数万円〜十数万円のレンジとみられる）¹⁰ | 放射温度測定・-20〜400℃対応。ドローン不要 |

**分かったこと**: 「敷地が狭いから安いドローンで済む」という予測は、機体の飛行性能（長距離・大容量バッテリー等）の面では正しいが、**サーマルセンサー自体の値段は敷地の広さと関係なく高い**（放射温度測定対応の製品は60万円台が最低ライン）。

**現実的な始め方（モデルBの適用）**: 低圧発電所はテニスコート1〜2面分と小さく、歩いて一周できる規模のため、**最初はドローンを使わず、ハンディ型サーマルカメラ（スマホ装着型）で地上から撮影する**方が初期投資を抑えられる。契約件数が積み上がり、ドローンへの投資が正当化できる段階になってから機材をアップグレードする、という順番が現実的。

### Phase 2（同業O&M業者へのツール販売）の裏付け状況（2026-07-04追記）

- **人材不足は確認**: O&M技術者に求められるスキルが「電気主任技術者等の資格・IT(遠隔監視)・土木(草刈り)・物理(洗浄)」と多岐にわたり複雑すぎることが人材不足の一因と指摘されている¹¹。「AI・ロボティクス活用はもはや"あったら良い"ではなく"事業継続に必須"になっている」という記述もある¹¹。
- **競合不在は確認（範囲内）**: 既存のAI点検製品（SOLAR CHECK、ドローンアイ、AI SolarPower on IDX等）は大規模発電所・需要予測・蓄電池まで含む大きなシステム向けで、「低圧中心の中小O&M業者向けの安価なAIレポートツール」というポジションの競合は検索範囲では見当たらなかった¹²。
- **業界構造**: Smart Energy・develop等の大手数社が上位を占める一方、業界ランキング(Top60等)から中小のO&M業者も多数存在するとみられる¹³¹⁴。ただし正確な「低圧専門・中小業者の数」は未確認。
- **残る限界**: Model Cの「77%が欲しいが5%しか使えていない」に相当する、**O&M業界固有の統計（O&M業者自身がAI導入を望んでいるが実行できていない、という直接的な調査データ）は見つからなかった**。人材不足とAIの必要性は別々の情報源からの確認であり、「同じ会社内で同時に起きている」ことの直接的な裏付けにはなっていない。
