# C群 リアリティチェック ── 「予算済み×人ゼロ」は本当に"発注"になっているのか

作成日: 2026-07-02
きっかけ: B4の教訓（制度があっても客がいるとは限らない。**執行された件数を見るまで需要と呼ばない**）をC群に適用。
検証した急所: ①金は本当に発注に変わっているか ②発注は誰が取っているか（個人が入れるか）③地域林政アドバイザーは"よそ者の個人"を本当に雇うのか。

---

## 0. 結論（先に言う）

**C1（森林経営管理の市町村実務代行）は検証を生き延びた。むしろ強化された。** ただし3つの重要な修正がある。

| 検証結果 | 中身 |
|---|---|
| ✅ **金は実際に動いている** | 譲与税の活用額は **96→210→270→399→464億円**（R1→R5）と毎年急増。単年度で見ると**譲与額の9割前後が使われている**。「47%塩漬け」は初期3年間の古い数字だった |
| ✅ **仕事のパイプラインは太くて長い** | 意向調査は6年間で**118万ha**実施、**1,160市町村**が制度に取組中。意向調査→再調査→集積計画→施業と何年も続く |
| ✅ **人の不足はガチ（数字で確認）** | 取組市町村1,160に対し、地域林政アドバイザーは**わずか353名（219自治体）**。**約5倍のギャップ** |
| 🎁 **公式の営業リストが存在** | 林野庁が「**アドバイザー活用を希望する市町村の一覧表**」を毎年公表している。**どこに営業すればいいか国が教えてくれている状態** |
| ⚠️ 修正1：無資格ルートには"研修ゲート"がある | 「地域に精通する方」ルートは**林野庁研修（八王子・4日間）or 県の同等研修（3日間等）の受講が条件**。開催は年1回程度＝**タイミングを逃すと1年待ち** |
| ⚠️ 修正2：個人"元請"の実例は未確認 | 意向調査・踏査の委託の受け皿は森林組合・システム会社・コンサルの事例が中心。**個人事業主が元請で落札した公開事例は今回見つけられなかった** → 個人の現実経路は「アドバイザー雇用」か「下請け」 |
| ⚠️ 修正3：C2（事前伐採）・C3（ため池）は発注実態が未確認 | 制度・勧告はあるが、**個人が取れる形の発注**が存在する証拠はまだない。ヒアリングで確認するまで「未検証」扱いに格下げ |

> 一言でいうと：**B4は「客がいると思ったらいなかった」。C1は「客（市町村）が実在し、毎年464億円使っていて、人が5倍足りず、営業先リストまで公開されている」。** ただし入口は「委託の元請」ではなく「**アドバイザーとして雇われる**」が本命ルート。

## 1. 取れた事実

### 金は動いているか（林野庁 取組状況公表）¹

| 年度 | 譲与税の活用額 |
|---|---|
| 令和元年度 | 96億円 |
| 令和2年度 | 210億円 |
| 令和3年度 | 270億円 |
| 令和4年度 | 399億円 |
| **令和5年度** | **464億円** |

- 単年度の譲与額（R5は約500億）に対し**9割前後が活用**される水準まで改善。**「塩漬け47%」は2019〜21年の累計の話で、市町村は使い方を覚えてきた**。
- 会計検査院がR6に譲与税事業の検査報告を出しており²、**「ちゃんと使え」の圧力は制度として継続**＝発注は今後も途切れにくい。
- 含意：**「金があるのに使えない」から「使い始めたが、実行する人と受け皿が足りない」フェーズに移行**。あなたにとってはむしろ好条件（発注が実在する）。

### 仕事は続くのか（森林経営管理制度の進捗）³

| 指標 | 数字 |
|---|---|
| 制度に取り組む市町村 | **1,160市町村**（R6年度末） |
| 意向調査の実施面積 | 6年間で**約118万ha** |
| 回答者のうち市町村への委託希望 | **約4割** → その後の集積・整備に続く |

- 私有林人工林は全国約670万ha。**118万ha÷6年のペースならあと数十年分の仕事量**。意向調査は入口にすぎず、その後（所有者探索・境界・集積計画・施業）が全部続く。

### 人は足りているのか（地域林政アドバイザー制度）⁴

| 指標 | 数字 |
|---|---|
| アドバイザー数 | H29: 38名 → **R6: 353名（219自治体：8県211市町村）** |
| 取組市町村とのギャップ | 1,160市町村 vs 353名 ＝ **約5倍不足** |
| 財政措置 | 特別交付税で**1人あたり上限500万円**、市町村は措置率0.7（**実質3割負担で人を雇える**） |
| 営業リスト | **林野庁が「活用希望市町村一覧」を公表**（令和8年度・9年度以降の希望を取りまとめ）＝行き先は公開情報 |

### 入口の要件（ここが一番大事）⁴⁵

アドバイザーの対象者は次の**いずれか**：
1. 森林総合監理士／林業普及指導員試験合格者／技術士（森林）／林業技士／森林施業プランナー等の**有資格者**
2. **「地域に精通する方」で、林野庁が実施する研修（森林技術総合研修所＝八王子、R7年度は4日間）またはそれに準ずる研修（県実施の例：3日間）を受講する者**

→ **ルート2が開いている。** 山開拓で開業済み・中京圏の中山間地で活動している実績は「地域に精通」の説明として筋が通る。**ただし**：
- 「地域に精通」の判断は**雇う市町村次第**（客観基準ではない）
- 研修は**開催時期が限られる**（R7年度は2026年2月開催の例）。逃すと1年待ち → **県実施の「準ずる研修」（宮城県等が実施）の中京圏版があるか、愛知・岐阜・三重県の林務課に確認するのが先決**
- 競合はおそらく県林務OB・森林組合OB（**推定**。論文にアクセスできず未確認）。ただし353人しかいない上に高齢OB中心なら、**「若くて現場を歩けて、AIで資料化できる」は明確な差別化**

### 委託（元請）の現実⁶

- 林野庁の事例集にある意向調査・踏査の委託先は**森林組合・林業事業体・システム会社・建設コンサル**。
- **個人事業主が元請で落札した公開事例は今回の調査では見つけられなかった**（存在しない証明ではないが、期待しない方が安全）。
- → 個人の現実経路は3つ：**(a) アドバイザーとして市町村に雇用される（本命・特交措置で市町村も雇いやすい）／(b) 森林組合・受託法人の下請け（踏査・戸別訪問の実働）／(c) 県の人材バンク・サポートセンター登録**。

### C2（事前伐採）・C3（ため池）は？

- **C2**：総務省勧告は2025年3月＝制度の追い風はあるが、**市町村が個人に出す形の発注実態は未確認**。C1でアドバイザーとして入り込んだ後に「うちの市の事前伐採の所有者調査もやりますよ」と**中から仕事を作る**のが現実的。
- **C3**：防災重点ため池の点検・劣化状況評価の発注は**県→建設コンサル**の構図が中心とみられ、個人が直接取れる形の発注は**未確認**。市町村農政課ヒアリングまで「未検証」に格下げ。

## 2. 仮説 vs 事実（B4と同じ形式で）

| 元の仮説 | 実データ | 判定 |
|---|---|---|
| 予算が塩漬けで執行する人がいない | 単年度執行は9割前後まで改善。**発注は生まれている** | △ 半分崩れたが、**むしろ好材料**（需要が実在する証拠） |
| 林務職員ゼロ4割＝人が足りない | アドバイザー353名 vs 取組1,160市町村＝**5倍ギャップ** | ○ 生きている（数字で強化） |
| 個人が市町村に食い込む制度ルートがある | アドバイザー制度は実在・特交措置あり・**希望市町村リストまで公表** | ○ 生きている（想定以上） |
| 無資格でも入れる | **研修受講（年1回・4日間等）がゲート**。「地域に精通」は市町村の裁量 | △ 条件付きで生きている |
| 委託を個人で受注できる | 元請は組合・法人中心。**個人元請の実例未確認** | ✕ 崩れた → 雇用（アドバイザー）or 下請けに切替 |
| C2・C3も同じ構図で取れる | 発注実態が確認できず | ✕ 未検証に格下げ（C1経由で中から作る） |

## 3. C1の再定義（入口の設計変更）

| | 旧C1（検証前） | **新C1（データ反映後）** |
|---|---|---|
| 入口 | 市町村から業務委託を受注 | **アドバイザーとして雇用される（週数日・特交措置500万円枠）** ＋ 森林組合の下請け |
| 営業先 | 中山間の市町村に手当たり次第電話 | **林野庁公表の「活用希望市町村一覧」から中京圏を抽出**（公式リスト）＋ 愛知・岐阜・三重県庁林務課（準ずる研修の有無確認） |
| 最初の一手 | 電話ヒアリング | ①**県林務課に「準ずる研修」の有無を電話**（研修ゲートの突破口）②希望市町村一覧の中京圏を確認 ③森林組合2〜3件に下請けの口を聞く |
| 収入の形 | 委託フィー | **雇用の報酬（月額/日当）＋下請け実働**。※実額は市町村の公募（例：奥州市）で要確認 |
| C2/C3の扱い | 並列で営業 | **C1で中に入ってから、中で仕事を作る**（事前伐採の所有者調査・ため池点検を「ついで」で提案） |

**強み適合の再確認**：アドバイザーの実務は「意向調査の設計・所有者対応・現地踏査・台帳整備・計画づくりの補助」。**山を歩ける×写真/位置情報で記録できる×AIで台帳・資料を量産できる**は、高齢OB型アドバイザーに対する明確な差別化。ここは崩れていない。

## 4. DECISION.md への影響

- **C1：28点を維持**（②供給ギャップは「5倍不足」の実数で裏付け強化。⑤即金性は研修ゲートのぶん引き続き低め＝もともと★★の評価で織り込み済み）。
- **信頼度が変わった**：B4が「仮説段階」から「検証して27点」になったのに対し、**C1は「検証して28点」**。同じ28点でもD1は未検証。→ **検証済みで最高点はC1のみ**。
- C3（ため池）は本編の順位表で「発注実態 未検証」の注記を付ける（DECISIONの5候補には入っていないため表の変更なし）。
