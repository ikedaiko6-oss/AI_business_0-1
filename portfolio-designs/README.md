# ポートフォリオ デザイン案 5選

パトさんから来る想定のHP制作依頼に備えて、提案用のデザインサンプルを5種類用意。
日本向け1案 ＋ 海外（アメリカ・スウェーデン・イスラエル・アフリカ）のトレンドを参考にした4案。

すべて自己完結の静的HTML（外部依存なし）。ブラウザでそのまま開けば見られる。
`index.html` がギャラリー（一覧）になっているので、まずはそこから開くのがおすすめ。

## 各案の設計意図

### 01 — 和ミニマル（日本向け）
生成り紙のような背景色、墨・藍・朱の3色に絞った配色。縦書きのあしらいや円相（ensō）モチーフで
「静けさ」を演出。余白（間）を広く取り、写真より言葉と構成で信頼を語る、老舗・専門職向けの型。

### 02 — US ボールドSaaS（アメリカ参考）
ダーク背景 × 紫〜ピンク〜オレンジのグラデーション、大きく強いタイポグラフィ、ピル型CTAボタン。
実績を「数字（KPI）」で見せるセクションを用意し、Silicon Valley系スタートアップのランディングページの
文法をそのまま踏襲。スピード感・成果訴求を前面に出したい事業向け。

### 03 — 北欧スカンジナビアン・ミニマル（スウェーデン参考）
白地に淡いセージグリーン・テラコッタのアクセント、細い罫線と円形モチーフ。装飾を削ぎ落とし、
写真と余白のバランスで「機能美」を伝える構成。IKEA・Acne Studios的な抑制の効いたトーン。

### 04 — スタートアップ・ネイション テック（イスラエル参考）
黒基調にネオンシアンのアクセント、グリッド背景、ターミナル風のUIあしらい。イスラエルの
「Startup Nation」文化を象徴する、技術力・スピード・実行力を訴求するテック企業向けの構成。

### 05 — アフリカン・ヴァイブラント（アフリカ参考）
サンドベージュの背景に、テラコッタ・ゴールド・グリーン・プラムの原色に近い配色。冒頭のパターン帯
（テキスタイル柄を抽象化した縞）や、ハードシャドウのボタンで力強さと熱量を表現。個性・世界観を
前面に出したいブランド向け。

## 使い方（提案時）

1. `index.html` を開いてギャラリーで全体像を見せる
2. クライアントの業種・ブランドイメージに近い案を2〜3個に絞って深掘り
3. 採用した案をベースに、実際のロゴ・写真・コピーへ差し替えて本番用に仕上げる

※ すべてダミー文言・プレースホルダーのサンプルです。実案件用に個別カスタマイズする前提の「型」として使ってください。
