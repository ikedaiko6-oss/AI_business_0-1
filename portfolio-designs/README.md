# マヤ暦占いカフェ「KIN」— デザイン案 5選

パトさんから来る想定のHP制作依頼に備えて、**「マヤ暦占いカフェ」という1つの具体的な業態コンセプト**を、
5カ国のデザイン言語で作り分けたサンプル。日本向け1案 ＋ 海外（アメリカ・スウェーデン・イスラエル・アフリカ）
のトレンドを参考にした4案。

すべて自己完結の静的HTML（外部依存なし）。ブラウザでそのまま開けば見られる。
`index.html` がギャラリー（一覧）になっているので、まずはそこから開くのがおすすめ。

## コンセプトの軸（全案共通）

マヤの暦「ツォルキン暦」は260日で一巡し、生まれた日ごとに固有の **KIN番号・20種の「紋章」・13の「音」** が
割り当てられるとされる暦占い。この構造をそのままカフェの体験設計に落とし込んでいる。

- **占いメニュー**: KINベーシック鑑定／相性鑑定／年間サイクル鑑定 の3段階
- **ドリンクメニュー**: 紋章を巡る4色（赤・白・青・黄）に対応する4種のドリンク
- **来店の流れ**: 生年月日を伝える → KINを算出 → 一杯とともに読み解く → 一言を持ち帰る

この骨格を保ったまま、屋号・コピー・トーン・ビジュアル言語だけを国ごとに翻訳している。

## 各案の設計意図

### 01 — 銀河茶寮 KIN（日本向け）
円相（ensō）を「260日の環」の視覚的メタファーとして使用。紋章を茶寮の一杯（ほうじ茶ラテ、玄米茶など）に
ひもづけ、老舗の落ち着きの中で暦を通訳するトーン。

### 02 — KIN.（アメリカ参考）
Co-Star的な「コスミック×スタートアップ」の文法をそのまま採用。全文英語、KPI風の統計バー（260 Day Cycle /
20 Sacred Seals など）、ピル型CTA。ドリンク名も英語で紋章と対応させている。

### 03 — KIN & ro（スウェーデン参考）
「ro（静けさ）」というスウェーデン語をブランド名に取り込み、情報量を絞った北欧ミニマルの流儀で鑑定体験を
設計。円形モチーフがツォルキンの環と自然に重なる。

### 04 — KINOS（イスラエル参考）
ツォルキン暦を「3,000年前に書かれた最初のアルゴリズム」と位置づけ、ターミナルUI・API風のエンドポイント
名（/red-dragon など）でドリンクメニューを表現。Startup Nation的なスピード感と技術訴求を占いに転用。

### 05 — KIN Baraza（アフリカ参考）
「バラザ（人々が輪になって語り合う集いの場）」を店名に採用し、暦を囲んで物語を交わす体験として設計。
鮮やかな配色とテキスタイル調のパターン帯で、コミュニティ感・熱量を表現。

## 使い方（提案時）

1. `index.html` を開いてギャラリーで全体像を見せる
2. クライアントの世界観・ターゲット層に近い案を2〜3個に絞って深掘り
3. 採用した案をベースに、実際の店舗名・価格・住所・写真へ差し替えて本番用に仕上げる

※ 屋号・価格・住所・お客様の声はすべてサンプルです。ツォルキン暦の紋章・音の名称と260日周期は実際の
体系に基づいていますが、鑑定内容の解釈は演出目的の一例であり、実占術としての正確性を保証するものでは
ありません。
